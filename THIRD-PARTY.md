# Third-party packages

This branch redistributes unmodified installers downloaded from official vendor release channels.

- Tailscale: https://tailscale.com/
- RustDesk: https://github.com/rustdesk/rustdesk
- Win32-OpenSSH: https://github.com/PowerShell/Win32-OpenSSH

Review each upstream project's license and terms before redistribution or production deployment. Tailscale package use is also subject to Tailscale's terms of service.
